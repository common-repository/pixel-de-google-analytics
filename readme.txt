Agrega fácilmente el seguimiento de Google Analytics (gtag.js) en tu web y configura parámetros básicos como los informes de datos demográficos o la atribución de enlace mejorada.

== Description ==

Inserta de forma rápida y sencilla el seguimiento de Google Analytics en todas las páginas de tu web y configura además parámetros básicos como la anonimización de IP, los informes de datos demográficos y de intereses o la atribución de enlace mejorada en unos pocos clics.

= Características =
* Compatible con las versiones más recientes de Wordpress.
* Sencillo y fácil de configurar, en apenas unos segundos dispondrás e Universal Analytics en tu web. 
* Posibilidad de activar/desactivar opciones adicionales como anonimizar IP, informes de datos demográficos y de intereses o la atribución de enlace mejorada.
* Ocupa menos de 20 Kb para interferir al mínimo en el rendimiento de tu web.
* Todos los detalles del plugin y un tutorial en <a href="https://www.labschool.es/implementar-google-analytics-en-wordpress/" target="_blank">Lab School</a>


== Installation ==

1. Puedes:
 * Subir la carpeta al directorio */wp-content/plugins/* vía FTP. 
 * Subir el archivo ZIP completo vía *Plugins -> Añadir nuevo -> Subir* en el Panel de Administración de tu instalación de WordPress.
 * Buscar **Google Analytics para Wordpress** en el buscador disponible en *Plugins -> Añadir nuevo* y pulsar el botón *Instalar ahora*.
2. Activar el plugin a través del menú *Plugins* en el Panel de Administración de WordPress.
3. Configurar el plugin dentro de *Ajustes -> Google Analytics*.
4. ¡Eso es todo!
